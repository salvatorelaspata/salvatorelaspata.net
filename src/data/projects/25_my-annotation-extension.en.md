# Personal Annotation

**Category:** Chrome Extension
**Technologies:** JavaScript, CSS, Chrome Extensions API

## Description

A Chrome extension for **annotating web content** with persistent local storage. Lets you highlight text and draw shapes directly on web pages, with annotations saved per page and persistent across sessions.

## Features

- **Text highlighting** with customizable colors
- **Drawing tools:** rectangle, circle, line
- **Annotation management:** view, edit, and delete
- **Color customization** for highlights and shapes
- **Persistent local storage** via browser localStorage
- **Multi-page support** with per-page annotation clearing
- Compatible with Chrome, Firefox, Edge, and Safari
