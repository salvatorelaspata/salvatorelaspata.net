# SAPUI5 Explorer

**Categoria:** Chrome Extension
**Tecnologie:** JavaScript, Chrome Extension Manifest V3, SAPUI5, OpenUI5

## Descrizione

Estensione Chrome per analizzare in profondità applicazioni **SAPUI5/OpenUI5** in esecuzione nel browser. Pensata per sviluppatori SAP che necessitano di ispezionare la struttura runtime di un'applicazione UI5 senza accedere al codice sorgente.

Compatibile con **UI5 1.x** (dalla 1.90) e **UI5 2.x** con compatibility layer.

## Funzionalità

- **Analisi framework:** versione, tema, lingua, debug mode, RTL, accessibilità
- **Ispezione componenti:** componenti registrati con metadati del manifest
- **Modelli dati:** tipologia e URL dei servizi OData
- **Gerarchia view e controller:** con metodi custom
- **Property bindings:** raggruppati per modello (30+ proprietà)
- **Log messaggi:** MessageManager e log UI5
- **Info runtime:** device, OS, orientamento, touch support
- **Tema e font:** correnti a runtime
- **Integrazione Fiori Launchpad:** info utente quando disponibile
- **Analisi URL:** Fiori intents, parametri, flag tecnici
- **Esportazione JSON:** copia dati completi negli appunti
