# sql-odata

**Category:** Prototype
**Technologies:** Node js, Fastify, Platformatic

## Description

An idea to create a Fastify plugin or a possible Platformatic integration that exposes relational database entities using the oData v2 protocol, used as a standard by SAP Fiori applications.
