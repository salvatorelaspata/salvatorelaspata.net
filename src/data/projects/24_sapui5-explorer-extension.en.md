# SAPUI5 Explorer

**Category:** Chrome Extension
**Technologies:** JavaScript, Chrome Extension Manifest V3, SAPUI5, OpenUI5

## Description

A Chrome extension for in-depth analysis of **SAPUI5/OpenUI5** applications running in the browser. Designed for SAP developers who need to inspect the runtime structure of a UI5 application without accessing the source code.

Compatible with **UI5 1.x** (from 1.90) and **UI5 2.x** with compatibility layer.

## Features

- **Framework analysis:** version, theme, language, debug mode, RTL, accessibility settings
- **Component inspection:** registered components with manifest metadata
- **Data models:** types and OData service URLs
- **View & controller hierarchy:** with custom methods
- **Property bindings:** grouped by model (30+ properties checked)
- **Message logging:** MessageManager messages and UI5 log entries
- **Runtime detection:** device info, OS, orientation, touch support
- **Theming data:** current theme and font information
- **Fiori Launchpad integration:** user info when available
- **URL analysis:** Fiori intents, query parameters, technical flags
- **JSON export:** copy full analysis data to clipboard
