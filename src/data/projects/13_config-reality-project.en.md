# Config Reality

**Category:** Project
**Technologies:** React JS, Three JS

## Description

The initial idea was to create a configurator for "teste di moro" (Moorish heads - being from Caltagirone, the city of ceramics) in 2D by overlaying photos with various components.. then I discovered the magical world of Three.js…
