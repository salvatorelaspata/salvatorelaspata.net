# Personal Annotation

**Categoria:** Chrome Extension
**Tecnologie:** JavaScript, CSS, Chrome Extensions API

## Descrizione

Estensione Chrome per **annotare contenuti web** con storage locale persistente. Permette di evidenziare testo e disegnare forme direttamente sulle pagine web, con le annotazioni salvate per pagina e persistenti tra le sessioni.

## Funzionalità

- **Evidenziazione testo** con colori personalizzabili
- **Strumenti di disegno:** rettangolo, cerchio, linea
- **Gestione annotazioni:** visualizza, modifica ed elimina
- **Personalizzazione colori** per highlights e forme
- **Storage locale persistente** tramite browser localStorage
- **Supporto multi-pagina** con pulizia annotazioni per pagina
- Compatibile con Chrome, Firefox, Edge e Safari
