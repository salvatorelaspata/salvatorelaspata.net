# sql-odata

**Categoria:** Prototype
**Tecnologie:** Node js, Fastify, Platformatic

## Descrizione

Idea di realizzare un plugin fastify o una possibile integrazione Platformatic che permette di esporre con il protocollo oData v2, utilizzato come standard dalle applicazioni SAP Fiori, le entità di un database relazionale.
